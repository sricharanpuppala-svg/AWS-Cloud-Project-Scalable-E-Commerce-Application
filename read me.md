Project Overview
This project is a Python-based simulation of a scalable e-commerce application inspired by AWS cloud architecture concepts. Built in a Jupyter Notebook, it demonstrates core e-commerce operations such as:
•	Product inventory management
•	Customer order processing
•	Sales and revenue reporting
•	Low-stock monitoring
•	Simulated AWS cloud service status tracking
•	Basic application performance metrics
The project is designed as an educational portfolio piece to showcase data processing, analytics, and cloud-inspired monitoring using pandas and numpy.
Features
Product Inventory System
•	Stores product catalog details include:
1.	Product ID
2.	Product Name
3.	Category
4.	Price
5.	Available Stock
Customer Order Management
•	Tracks customer purchases
•	Calculates total order values automatically
•	Generates sales reports
Revenue Analytics
•	Computes total revenue generated from all orders
•	Provides business performance insights
Low Stock Alerts
•	Identifies products with stock below threshold
•	Helps simulate inventory management workflows
AWS Cloud Services Simulation
•	Simulates status logs for:
EC2 Instance
RDS Database
CloudFront CDN
Performance Monitoring
•	Generates random API response times
•	Evaluates average system responsiveness
•	Flags optimization status
